### Project Title
Bikeshare Module

### Description
This module explores data related to bike share systems for three major cities in the United States: Chicago, New York City, and Washington.

### Files used
The bikeshare.py script used the following files as input:

1: "chicago.csv"

2: "new_york_city.csv"

3: "washington.csv"
